# Partner HTML Style Defaults

Use this visual style for optional partner API HTML guides when the SA accepts the organization default.

## Brand / Theme

- Brand text: `freewillsolutions`
- Primary color: gold `#C8920A`
- Secondary gold: `#E5A812`
- Pale gold background: `#FDF6E3`
- Dark text: `#1A1410`
- Warm background: `#FAFAF8`
- Section background: `#F7F4EE`
- Border line: `#EDE8DF`
- Success green: `#1A7A4A`
- Error red: `#A03020`
- Info blue: `#1A4A8A`
- Warning amber: `#8A5A0A`

## Layout

- Sticky left sidebar navigation.
- Topbar with breadcrumb, API version, and status chip.
- Hero section with project name, API version, base path, auth method, protocol, and format.
- Numbered content sections.
- Endpoint cards for each API.
- Tables for headers, parameters, response fields, errors, and data protection.
- Dark code blocks for JSON/cURL examples.

## Required Sections

- Introduction / Overview
- Base URL
- Request Flow
- Authentication
- Request Headers
- Endpoints
- Response Headers
- Data Protection
- Error Codes
- Request Examples

## Safety

Never embed real API keys, production secrets, private credentials, or sensitive sample personal data in the HTML.
